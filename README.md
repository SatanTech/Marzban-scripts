# Marzban-scripts
Scripts for Marzban

## Install
Install Marzban on your server using this command
```bash
sudo bash -c "$(curl -sL https://satantech.serv00.net/Marzban-Script/marzban.sh)" @ install
```

Or you can only install this script (marzban command) on your server by using this command
```bash
sudo bash -c "$(curl -sL https://satantech.serv00.net/Marzban-Script/marzban.sh)" @ install-script
```
